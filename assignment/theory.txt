25.What is List? How will you reverse a list?

    In Python, a List is a versatile, built-in data structure used to store a collection of items in a specific order. Lists are mutable, 
    meaning you can change, add, or remove elements after the list has been created.
    To reverse a list, you can use several methods depending on whether you want to modify the original list or create a new one:
    1. Reversing In-Place (Modifies the original list) :
    If you don't need to keep the original order, use the built-in list.reverse() method.
    Ex:
        fruits = ["apple", "banana", "cherry"]
        fruits.reverse()
        print(fruits) # Output: ["cherry", "banana", "apple"]
    It is highly memory-efficient because it does not create a copy.
    2. Slicing (Creates a new list)
    The slicing trick [::-1] is the fastest and most concise way to create a reversed copy while keeping the original list intact. 
    Ex :
        numbers = [1, 2, 3, 4]
        reversed_numbers = numbers[::-1]
        print(reversed_numbers)  # Output: [4, 3, 2, 1]
    3. Using the reversed() Function
    The reversed() function returns a reverse iterator. 
    This is ideal for looping through a list backwards without wasting memory on a new list. 
    Ex :
    for item in reversed(["a", "b", "c"]):
        print(item)  # Prints: c, b, a

28.Differentiate between append () and extend () methods?

    In Python, both append() and extend() are used to add elements to a list, but they handle their input differently: 
    append(): Adds its argument as a single element at the end of the list. 
    Even if you pass another list, it remains nested.
    extend(): Iterates over its argument (which must be an iterable, like a list, tuple, or string) and adds each item individually.
    Ex :
        list1 = [1, 2]
        list1.append([3, 4])
        # Result: [1, 2, [3, 4]] (Nested list)

        list2 = [1, 2]
        list2.extend([3, 4])
        # Result: [1, 2, 3, 4] (Flattened list)

67. How can you pick a random item from a range?

    To pick a random item from a range in Python, you can use the random.choice(), random.randrange(), or random.randint() functions from the built-in random module. 

    1. Using random.choice()
    The random.choice() function is a straightforward way to pick a single random element from a sequence, including a range object. 

    import random
    # Define your range (e.g., from 0 up to, but not including, 10)
    my_range = range(0, 10)

    # Pick a random item
    random_item = random.choice(my_range)
    print(f"The random item is: {random_item}")

    2. Using random.randrange()
    The random.randrange() function is optimized for generating a random number from a specified range and supports an optional step value. The stop value is exclusive. 
    
    import random
    # Generate a random integer from 0 up to (but not including) 10
    random_int = random.randrange(0, 10)

    print(f"The random integer is: {random_int}")
    You can also specify a step, for example, to get a random even number: 

    # Generate a random even integer from 0 up to (but not including) 10
    random_even = random.randrange(0, 10, 2)
    print(f"The random even number is: {random_even}")

    3. Using random.randint()
    The random.randint() function is an alias for random.randrange(a, b+1) and provides a random integer within an inclusive range (both the start and end values are possible outcomes). 

    import random
    # Generate a random integer between 0 and 10, inclusive
    random_inclusive = random.randint(0, 10)
    print(f"The random inclusive integer is: {random_inclusive}")

68. How can you get a random number in python?

    In Python, you can generate random numbers using the built-in random module, which provides functions for various needs, from simple integers and floats to specific statistical distributions. 

    Key Functions in the random Module
    You must first import the module: 

    import random

    Then, you can use the following functions:

    1. random.randint(a, b): Returns a random integer N such that a <= N <= b (both endpoints are included).
    python
    # Example: rolling a six-sided die (random number between 1 and 6, inclusive)
    roll = random.randint(1, 6)
    print(roll)

    2. random.randrange(start, stop, step): Returns a randomly selected element from range(start, stop, step).
    python
    # Example: a random even number between 0 and 10 (0, 2, 4, 6, 8, 10)
    even_number = random.randrange(0, 11, 2)
    print(even_number)

    3. random.random(): Returns a random floating-point number in the range (0.0, 1.0) (inclusive of 0.0, but exclusive of 1.0).
    python
    # Example: a random float between 0 and 1
    rand_float = random.random()
    print(rand_float)

    4. random.uniform(a, b): Returns a random floating-point number N such that a <= N <= b.
    python
    # Example: a random float between 10.5 and 25.5
    rand_float_range = random.uniform(10.5, 25.5)
    print(rand_float_range)
